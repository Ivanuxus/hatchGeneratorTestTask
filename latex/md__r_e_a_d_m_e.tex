Тестовое задание\+: Генерация штриховки для SLM-\/слайсера (C++20) Описание задачи В этом тестовом задании необходимо разработать модуль для генерации штриховки (hatch) внутри заданного контура квадратной или прямоугольной детали. Программа должна\+:
\begin{DoxyItemize}
\item Получать на вход массив точек, описывающих контур детали.
\item Создавать линии штриховки под указанным углом и с заданным шагом. Позволять выводить результат либо в виде текстового списка координат, либо в виде визуализации. Требования к функционалу
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item Входные данные Программа должна принимать на вход массив точек, описывающий простой квадрат или прямоугольник, угол наклона штриховки (например, 0–180°) и шаг между линиями штриховки (например, в мм). Пример массива точек\+: struct Point\+\_\+2 \{ double x; double y; \}; std\+::vector$<$std\+::vector$<$\+Point\+\_\+2$>$$>$ contours\+Points = \{ \{ \{0,0\}, \{10,0\}, \{10,10\}, \{0,10\} \} \};
\item Генерация штриховки
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Разработать функцию, которая генерирует линии под указанным углом внутри контура квадрата(прямоугольника).
\item Линии должны располагаться с заданным шагом.
\item Линии можно хранить в массиве (например, std\+::vector$<$\+Line\+\_\+2$>$).
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item Вывод результата Программа обязана предоставлять текстовый вывод. Дополнительно, по желанию, можно реализовать визуализацию квадрата и линий штриховки (например, используя SVG).
